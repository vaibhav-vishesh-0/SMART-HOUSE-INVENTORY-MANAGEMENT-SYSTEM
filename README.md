# Smart House Inventory Management System

A Python-based application to manage household inventory efficiently. It allows users to store, update, and organize item details in a simple and structured way.

## Features

* Add new items
* View all items
* Update item details
* Delete items
* Search items

## Technologies Used

* Python
* CSV File Handling

## How to Run

1. Clone the repository
   git clone https://github.com/vaibhav-vishesh-0/SMART-HOUSE-INVENTORY-MANAGEMENT-SYSTEM.git

2. Open the project folder

3. Run the program
   python main.py

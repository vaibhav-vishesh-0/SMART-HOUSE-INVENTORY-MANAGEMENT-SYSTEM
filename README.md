🏠 Smart House Inventory Management System

This project is a Smart House Inventory Management System developed using Python. It helps users efficiently manage household items by allowing them to store, update, and track inventory data in a structured way.

The system provides features like adding new items, viewing existing inventory, updating details, and deleting records. It uses file handling techniques (such as CSV/JSON) to ensure data persistence and easy retrieval.

The main objective of this project is to simplify home inventory management and demonstrate the use of core programming concepts like file handling, data structures, and modular programming.
